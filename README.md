# qubership-apihub-build-task-consumer

NodeJS microservice for APIHUB Package versions builds.

## Installation

```bash
$ npm
```

## Running the app

```bash
# development
$ npm start

# watch mode
$ npm start:dev

# production mode
$ npm start:prod
```

## Test

```bash
# unit tests
$ npm test

# e2e tests
$ npm test:e2e

# test coverage
$ npm test:cov
```
